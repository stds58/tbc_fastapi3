
Что делает моржовый оператор?
Моржовый оператор (:=) позволяет присвоить значение переменной внутри выражения.
Это особенно полезно, когда нужно использовать значение переменной сразу после её вычисления,
не создавая дополнительных строк кода.

Пример без моржового оператора:
username = payload.get("sub")
if not username:
    return None

С моржовым оператором:
if not (username := payload.get("sub")):
    return None
